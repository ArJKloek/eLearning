Een vereenvoudigde weergave van het processchema voor \(\ce{SO3}\) (grondstof voor de productie van zwavelzuur) is te zien in onderstaande figuur. Zwavel wordt verbrand in aanwezigheid van 100 % overmaat lucht in de verbrandingsoven. Voor de reactie:

$$\ce{S + O2 -> SO2}$$

wordt een conversie van 90 % bereikt. In de converter bedraagt de conversie van \(\ce{SO2}\) tot \(\ce{SO3}\) 95%.

a.  Bereken de grootte van de stroom L in kmol/hr.
b.  Bereken de grootte van de tussenstroom T.
c.  Bereken de grootte en samenstelling van de productstroom P (molfracties).

Basisformules: chemische reacties, component-molbalansen

Oplossingsplan: systematische aanpak via molbalansen (b + c)

Uitwerking:

a. 31.2 kmol S/h  ≡  31.2 kmol \(\ce{O2}/\text{h}\)
  \(\text{L bevat}\ 2\times 31.2\)  =  62.4 kmol \(\ce{O2}/\text{h}\) (100% overmaat)
   ≡  \((79/21) \times 62.4 =\) kmol \(\ce{N2}/\text{h}\)
  \(\text{L}=\text{L}_\ce{O2} + \text{L}_\ce{N2}\)  =  62.4 + 234.7 = kmol lucht/h
  N.B. \(\text{L}_\ce{N2} = \text{T}_\ce{N2}\)  =  \(\text{P}_\ce{N2} =\) kmol/h (zie b en c)

b. CM molbalansen over de oven:            
    Accumulatie  =  Invoer  \(-\)  Uitvoer  \(+\)  Productie  \(-\)  Omzetting
  \(\ce{O2}\) 0  =  \(\text{L}_\ce{O2}\)  \(-\)  \(\text{T}_\ce{O2}\)  \(+\)  0  \(-\)  \(\text{X} \cdot \text{F}_\text{S}\)
    0  =  62.4  \(-\)  \(\text{T}_\ce{O2}\)  \(+\)  0  \(-\)  \(0.90 \times 31.2\)
                    \(\text{T}_\ce{O2} =\) kmol/h
                     
  \(\ce{SO2}\) 0  =  0  \(-\)  \(\text{T}_{\ce{SO2}}\)  \(+\)  \(\text{X} \cdot \text{F}_\text{S}\)  \(-\)  0
    0  =  0  \(-\)  \(\text{T}_{\ce{SO2}}\)  \(+\)  \(0.90 \times 31.2\)  \(-\)  0
                    \(\text{T}_{\ce{SO2}} =\) kmol/h
                     
  \(\text{N}_{2}\)                 \(\text{T}_{\ce{N2}} =\) kmol/h
