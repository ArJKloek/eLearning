Men heeft een reactor waarin men 100 kmol verse reactant A per tijdseenheid binnenvoert. In de reactor vinden de volgende reacties plaats:

$$\ce{A->B}\quad (\text{gewenst})$$

$$\ce{A->C}\quad (\text{niet gewenst})$$

Het mengsel dat uit de reactor komt wordt naar een scheider gevoerd waar het wordt gescheiden in A, B en C. Stof A wordt weer naar de reactor teruggevoerd.

Bereken alle stofstromen als de conversiegraad \(\text{X}_{\text{s}} =0.80\), de selectiviteit \(\text{S} = 0.70\) en alle niet omgezette A wordt teruggevoerd.

Opmerking:

De hoeveelheid A die in de reactor wordt omgezet is hier gelijk aan de verse invoer. Dit geldt dus alleen als alle niet-omgezette A wordt teruggevoerd! 

Analyse:

Basisformules: Balansen over het geheel of een gedeelte van het systeem. 

Oplossingsplan: 

De hoeveelheid A die de reactor verlaat wordt in zijn geheel teruggevoerd en is dus gelijk aan R. Dit is echter ook gelijk aan wat over blijft van de totale reactorinvoer na omzetting. Er kan gewerkt worden met componentstromen.

Uitwerking:

1. Systeem: Reactor
  Componenten molbalansen (CM):
    Accumulatie  =  Invoer  \(-\)  Uitvoer  \(+\)  Productie  \(-\)  Omzetting    
  A 0  =  \(\text{F}_\text{IN}\)  \(-\)  \(\text{T}_\text{A}\)  \(+\)  0  \(-\)  \(\text{X}_\text{S}\cdot \text{F}_\text{IN}\)    
    0  =  100 + R  \(-\)  \(\text{T}_\text{A}\)  \(+\)  0  \(-\)  \(0.80 \cdot (100 + \text{R})\)    
    met R = \(\text{T}_\text{A}\)                 \(\text{T}_\text{A}\) = kmol/h
                      R = kmol/h
  B 0  =  0  \(-\)  \(\text{T}_\text{B}\)  +  \(\text{S} \cdot \text{X}_\text{S} \cdot \text{F}_\text{IN}\)  \(-\)  0    
    0  =  0  \(-\)  \(\text{T}_\text{B}\)  +  \(0.70 \cdot 0.80(100 + \text{R})\)  \(-\)  0    
                      \(\text{T}_\text{B}\) = kmol/h
  C 0  =  0  \(-\)  \(\text{T}_\text{C}\)  +  \((1 - \text{S}) \cdot \text{X}_\text{S} \cdot \text{F}_\text{IN}\)  \(-\)  0    
    0  =  0  \(-\)  \(\text{T}_\text{C}\)  +  \((1-0.70) \cdot 0.80(100 + \text{R})\)  \(-\)  0    
                      \(\text{T}_\text{C}\) = kmol/h
2. Systeem: de scheider
    \(\text{T}_\text{A} = \text{R}\); evenzo: \(\text{T}_\text{B} = \text{P}\) en \(\text{T}_\text{C} = \text{W}\)
